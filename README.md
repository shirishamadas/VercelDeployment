# Node.js Swagger Vercel API

This project provides a Node.js server with GET and POST API endpoints, Swagger UI for API testing, and is ready for Vercel deployment.

## Setup

1. Install dependencies:

   ```sh
   npm install
   ```

2. Start the server locally:

   ```sh
   npm start
   ```

3. Open Swagger UI to test APIs:
   - Visit [http://localhost:3000/api-docs](http://localhost:3000/api-docs)

## Deployment (Vercel)

1. Push your code to GitHub.
2. Go to https://vercel.com, import your repo, and deploy.

## API Endpoints

- **GET** `/api/hello`
- **POST** `/api/echo`

See Swagger UI for details and testing.
